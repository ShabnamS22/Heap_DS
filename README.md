## implementation 
## HeapSort
- Build a Max heap
- Repeatedly extract the maximum element
- Time complexity best O(n log n), average O(n log n), worst O(n log n).
- Space Complexity O(1) auxilary space. inplace sorting, no additional arrays used.

## Theoretical Comparison
- HeapSort: Best case O(n log n), average case O(n log n), worst case O(n log n), space O(1).
- QuickSort: Best case O(n log n), average case O(n log n), worst case O(n^2), space O(log n)
- MergeSort: Best case O(n log n), aveage case O(n log n), worst case O(n log n), space O(n).
